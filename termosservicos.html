2

Descrição do serviço

O Gabriel é um assistente pessoal inteligente que funciona como Progressive Web App (PWA). O aplicativo oferece as seguintes funcionalidades:

  • Assistente de IA conversacional via API Groq (modelo LLaMA)
  • Agenda pessoal com lembretes e notificações push
  • Controle financeiro pessoal
  • Caderno de notas e organização em pastas
  • Integração opcional com Gmail e Google Agenda
  • Previsão do tempo via serviço wttr.in
  • Memória automática de contexto das conversas
devices
Dados locais: Todos os dados são armazenados no seu dispositivo. O Gabriel não possui servidores de armazenamento de dados de usuários.
3

Integração com serviços Google

A integração com Gmail e Google Agenda é completamente opcional. Ao autorizar o acesso, você concorda que o Gabriel poderá:

  • Ler e-mails da sua caixa de entrada para exibição e resumo no app
  • Enviar e-mails em seu nome quando explicitamente solicitado
  • Marcar e-mails como lidos quando você solicitar
  • Ler eventos do Google Agenda para exibição no painel
  • Criar eventos no Google Agenda quando você solicitar via chat

O uso dos dados obtidos pelas APIs do Google está em conformidade com a Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de Uso Limitado.

Você pode revogar o acesso a qualquer momento em myaccount.google.com/permissions.

security
Uso Limitado: O Gabriel não usa dados do Google para publicidade, treinamento de IA, ou qualquer finalidade além das explicitamente listadas e autorizadas pelo usuário.
